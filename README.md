## first npm package

### my-test-package-one

## installation: 
``` js 
    npm i my-test-package-one
``` 

## Usage:
* CommonJS Syntax
```js 
    const {addition,multiply} = require("my-test-package-one")

    addition(2,3)
    multiply(3,4)
```
* ES6 Syntax
```js 
    import  {addition,multiply} from "my-test-package-one"

    addition(2,3)
    multiply(3,4)
```
